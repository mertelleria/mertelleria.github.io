Pospolo

Set de envases de vidrio diseñados para contener fósforos de incienso, cada color y textura escogidos representa un aroma.

Lo que más valoro de este proyecto es la interacción que tuve al contactar con un profesional para discutir la viabilidad. El artesano del vidrio requirió bocetos, maquetas en 3D, y planos CAD.

Por otro lado, con fin de darle más autenticidad, construí mi primera página web (utilizando HTML y CSS) para el producto, simulando una web de compra-venta.
